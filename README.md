# Claude Code Engineering Workshop — Hands-On Labs

Companion repository for the **Claude Code Engineering in Practice** workshop.

## Prerequisites

- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) installed
- Node.js 20+ and pnpm
- An Anthropic API key (set as `ANTHROPIC_API_KEY`)

## Workshop Structure

| Module | Directory | Topic |
|--------|-----------|-------|
| 1 | `01-memory/` | Memory System — CLAUDE.md |
| 2 | `02-skills/` | Skills — On-Demand Expertise |
| 3 | `03-subagents/` | Sub-Agents — Divide and Conquer |

## Quick Start

```bash
git clone https://github.com/huangjia2019/claude-code-engingeering-EN.git
cd claude-code-engingeering-EN
```

Then follow the exercises in each module's `README.md`.
